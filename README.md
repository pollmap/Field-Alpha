# Field Alpha

> **발로 뛴 투자 인사이트** — 전국 산업단지·신도시·기업 현장 답사 기록을 체계화한 투자 리서치 아카이브

## Overview

Field Alpha는 2023년 12월부터 시작된 전국 산업단지, 신도시, 기업 현장 답사 기록을 체계적으로 관리하는 투자 리서치 플랫폼입니다.

### Features
- **인터랙티브 지도**: 전국 답사 지점을 Leaflet 지도에 시각화
- **산업 클러스터 분석**: 송도 바이오, 평택 반도체 등 클러스터 단위 분석
- **관찰/인사이트 분리**: 객관적 사실과 투자 의견을 명확히 구분
- **시계열 추적**: 동일 장소 재방문 시 변화 기록
- **통합 검색**: Fuse.js 기반 퍼지 검색

## Tech Stack
- Next.js 14 (App Router, Static Export)
- TypeScript + Tailwind CSS
- Leaflet (Interactive Map)
- Recharts (Charts)
- GitHub Pages

## Getting Started

```bash
npm install --legacy-peer-deps
npm run dev
```

## Build & Deploy

```bash
npm run build  # Static export → out/
```
